 贪吃蛇游戏（Snake Game）  

这是一个使用 Java + Swing 实现的贪吃蛇小游戏，支持完整游戏逻辑、蛇身方向贴图、食物生成、分数计数及可打包成客户端的可执行 Jar 文件。  

 功能特点

- ✅ 游戏主循环使用 `Timer` 驱动  
- ✅ 蛇头、蛇身、蛇尾方向贴图  
- ✅ 蛇增长机制：吃到食物增加长度，普通移动尾部移动  
- ✅ 食物生成不出现在蛇身上，格子对齐  
- ✅ 游戏状态管理：
  - 开始界面（Press SPACE to Start）  
  - 运行状态  
  - 暂停（SPACE 切换）  
  - 游戏结束界面（Game Over + Restart）
  - 使用箭头控制蛇的前进方向
- ✅ 分数随蛇长度增长，速度随分数提升  
- ✅ 支持最高分本地保存（txt）
